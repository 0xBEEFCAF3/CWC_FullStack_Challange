# To run
1) clone repo
2) cd into repo
3) npm install
4) npm start
5) Open https://localhost:8080/ <b> Make sure it is https </b>

## Application has been tested in Chrome 70.0, Firefox 65.0b3, and Microsoft Edge 42.17.
